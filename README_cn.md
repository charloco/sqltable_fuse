# 介绍

基于fuse将数据库段内容暴露出来,用文件的方式来修改内容,这样核心配置(例如yaml格式的)和动态脚本就可以存储在数据库中。

适合规模不大的场景, 这重场景只需要维护一个数据库集群就可以了,不需要对象存储了。

有了这个工具，就可以更舒服的修改相应的配置和脚本了
